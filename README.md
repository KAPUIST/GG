# MuscleFormula Feature

- 본인의 운동 기능을 기록할 수 있습니다. 
- 운동기록이 높은 상위 랭크의 유저를 확인할 수 있습니다. 
- 마음에 드는 운동 기록 게시판을 좋아요 클릭하여 따로 볼 수 있습니다.
- 운동 게시물을 기록하고, 공유할 수 있습니다.  
- 지도를 통해 근처의 헬스장을 확인할 수 있습니다. 
- 채팅을 통해 단체 운동을 진행 할 수 있습니다.
